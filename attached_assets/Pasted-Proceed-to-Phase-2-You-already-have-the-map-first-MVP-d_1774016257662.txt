### 3) Privacy enforcement
This is mandatory:

- Need posts must never expose exact private coordinates or exact address via public API responses
- Public map responses must only use publicLat/publicLng
- For need posts, publicLat/publicLng must represent fuzzed/approximate district-level coordinates only
- NGOs may expose public exact coordinates
- If needed, create server-side response mappers so unsafe fields are never returned

---

### 4) Active map filtering rules
Public map-facing responses must return only:
- status = active
- expiresAt > now

Expired or inactive posts must be hidden by default.
